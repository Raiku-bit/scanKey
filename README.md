##  Future Features

- [ ] **⌨️ Special Keys Detection**
  - *Current state:* Only detects standard alphabetic characters.
  - *Planned:* Expand capabilities to capture modifier keys (`Ctrl`, `Alt`, `Shift`), arrows, and system shortcuts.
- [ ] **🌐 Built-in Web Server**
  - *Current state:* Local storage only.
  - *Planned:* Implement an automated, self-hosted web server within the application to easily expose, transfer, and manage saved data.
- [ ] **🔄 Autorun (Boot on Startup)**
  - *Current state:* Manual execution required.
  - *Planned:* Add support for automatic background startup when the operating system boots.
